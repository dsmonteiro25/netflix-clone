# Clone da Interface da Netflix 🎬

Este projeto é uma recriação da interface da página inicial e da tela de login da Netflix, com foco em responsividade, estilo moderno e interatividade usando HTML, CSS e JavaScript puro.

## 📁 Estrutura do Projeto

/
├── index.html # Página inicial com carrossel e motivos para assinar
├── login.html # Página de login com validação simulada
├── /css
│ ├── styles.css # Estilos principais da página inicial
│ ├── login.css # Estilos da tela de login
│ └── footer.css # Estilos compartilhados do rodapé
├── /js
│ ├── script.js # Scripts da página inicial (carrossel, animações, etc.)
│ └── login.js # Scripts de validação da tela de login
├── /assets
│ └── banner.jpg # Imagem de fundo (adicionar manualmente)


## 🔧 Funcionalidades

### Página Inicial (`index.html`)
- Hero section com título, descrição e campo para email
- Carrossel de filmes com navegação circular
- Animações de entrada e efeitos de hover
- Seção “Mais motivos para assinar” com cartões animados
- Rodapé estilizado com links e seletor de idioma

### Tela de Login (`login.html`)
- Inputs com labels flutuantes
- Validação básica de email/senha
- Simulação de login (sem backend)
- Link para criar conta e opção de "Usar código de acesso"
- Responsividade para dispositivos móveis

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (Grid, Flexbox, media queries)
- JavaScript (DOM Manipulation, Carousel, Validação de Formulário)
- Layout responsivo e adaptado para múltiplas resoluções
